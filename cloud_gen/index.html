This word cloud generator was created with data structures and object oriented programming techniques. It uses the standard java library to quickly record the number of times a certain word occurs in a text and filtering out certain punctuation. Then, the user determines the quantity of the top scoring words to be alphabatized and printed to an html page. During this process, the words are resized depending on their occurances in relation to the most and least frequently that resizes them. The words are also randomly colored during this process. In the end you can sort through novel length texts in a fraction of a second.
